# Gemini 2.5 Flash Lite — P1 paired pilot

Original study-centered files are retained at `Gemini2.5Flash-Lite/P1`.

- `Ideas_Data_To_Shuffle/`: source M0/M1/M3 concept artifacts.
- `InputJSON_*`: blinded evaluation inputs; public inputs remain here.
- `Output_*`: archived judge outputs.
- `Results/Validation/`: validator audits.
- `Results/Analysis/`: unblinded analytical outputs.
- `Private/`: private mappings, logs, credentials, and/or participant records.

Do not publish anything inside `Private/` without a separate privacy and
secret-scan review.
