# Gemini 2.5 Flash-Lite P1 release scope

Status: archived legacy/exploratory study artifact.

This pre-human release includes public-safe legacy evaluator inputs, judge outputs, validation artifacts, manifests, and manuscript artifacts. It deliberately omits raw idea-card source files and unblinded analysis outputs until the planned human evaluation is complete.

The public builder removes `Private/`, raw debug/log folders, files with private-key-like names,
and absolute local paths. This folder is visible for provenance and historical completeness;
its contents must not be represented as part of the strict Gemini 3.1 Flash-Lite P1 main analysis
unless a manuscript explicitly states otherwise.
