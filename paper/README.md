# Paper Evidence Package

This repository contains both the manuscript and the result evidence used to support it:

- `main.tex` and `main.pdf`: main anonymous manuscript.
- `supplementary_results.tex` and `supplementary_results.pdf`: detailed statistical tables.
- `claim_evidence_registry.csv`: each allowed/prohibited manuscript claim mapped to its evidence.
- `../results/tables/`: machine-readable tables S01–S19 and main table matrices.

The main manuscript makes claims only from the Gemini 3.1 Flash Lite P1 individual and
directional protocols. Gemini 2.5 results are supporting/developmental. Neutral
results are fully retained but not used for efficacy claims.

## Reviewer-facing table map

- Main paper Tables 1–5 are compact evidence tables.
- Supplementary Tables S1–S13 are full aggregate statistics.
- `results/tables/TABLE_PROVENANCE_SHA256.csv` provides source-to-published table hashes.
