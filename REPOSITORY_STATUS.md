# Repository Status

**Version:** v0.9.3-prehuman-evidence-expanded  
**State:** private/reviewer-controlled pre-human evidence package

This version is suitable for internal audit, supervisor review, and controlled
reviewer access. It is not a final public v1.0 release because the planned human
evaluation has not yet been collected and frozen.

The package contains aggregate statistics that do not reveal candidate-level mappings.
Do not publish it broadly until you decide that doing so cannot affect the blinding of
human evaluators.
