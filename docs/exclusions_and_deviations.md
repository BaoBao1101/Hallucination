# Exclusions and deviations

Gemini neutral v1, v2, and v3 are excluded from strict neutral inference because their model-authored metric-level preference labels did not satisfy the pre-specified score-to-preference consistency rule. No manual repair was performed.

The strict neutral paired results in this public release are therefore single-judge LLM evidence based on ChatGPT v1. Individual evaluation and directional traceability contain strict-valid outputs from both archived judges.
