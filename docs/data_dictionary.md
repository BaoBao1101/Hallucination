# Data dictionary

- `Public_Analysis_Ready/individual_scores_long.csv`: candidate-level individual scores.
- `Public_Analysis_Ready/directional_scores_long.csv`: directional traceability scores.
- `Public_Analysis_Ready/directional_prototype_preferences.csv`: directional prototype choices.
- `Public_Analysis_Ready/neutral_chatgpt_scores_long.csv`: included ChatGPT neutral M1/M3 score pairs.
- `Public_Analysis_Ready/neutral_chatgpt_preferences_long.csv`: included ChatGPT neutral preference labels mapped to M1/M3/TIE.

`M0_DIRECT_LOW_TEMP` is exploratory. The main paired transformation comparison is M1 versus M3.

Legacy Gemini 2.5 Flash-Lite P0/P1 artifacts are documented separately in `manifests/legacy_study_registry.csv`.
